# Sonar Rock vs Mine Prediction

# Project Overview
This project predicts whether an underwater object is a **Rock** or a **Mine** using Machine Learning. The model is trained on the Sonar dataset using Logistic Regression.

## 🎯 Objective
To classify sonar signals into two categories:
- Rock (R)
- Mine (M)

## 📂 Dataset
- Dataset Name: Sonar Dataset
- Total Records: 208
- Total Features: 60
- Target Classes:
  - R (Rock)
  - M (Mine)

## 🛠 Technologies Used
- Python
- Google Colab
- NumPy
- Pandas
- Scikit-learn
- Pickle



## 🤖 Machine Learning Algorithm
- Logistic Regression

# Project Workflow
1. Import Libraries
2. Load Dataset
3. Data Preprocessing
4. Train-Test Split
5. Train Logistic Regression Model
6. Evaluate Accuracy
7. Predict Rock or Mine
8. Save Model using Pickle

## 📁 Project Files

- Sonar_Rock_vs_Mine_Prediction.ipynb
- sonar_data.csv
- trained_model.pkl
- README.md
- requirements.txt

## 🚀 How to Run

1. Clone this repository.
2. Install required libraries: pip install -r requirements.txt
3. Open the Jupyter Notebook or Google Colab.
4. Run all cells.
5. Use the trained model for prediction.



## 📊 Output

The model predicts whether the sonar signal belongs to:
- Rock
- Mine
